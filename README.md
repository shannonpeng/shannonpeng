# shannonpeng.com
new personal website :o

install less and less-watch-compiler (I have it installed globally):
```
npm install -g less
npm install -g less-watch-compiler
```
set up virtualenv:
```
python3 -m venv venv
```
activate venv and install python packages:
```
. venv/bin/activate
pip install -r requirements.txt
```
then run locally:
```
sh start.sh
```
